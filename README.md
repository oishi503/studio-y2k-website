# Studio Y2K Website (https://oishi503.github.io/studio-y2k-website/)

A static fashion landing page inspired by early 2000s (Y2K) aesthetics.

This project focuses on layout design, typography, and visual storytelling using HTML and CSS.

## Features
- Responsive landing page layout
- Navigation bar with section links
- Promotional banners and call-to-action buttons
- Category-based layout for products

## Tech Stack
- HTML
- CSS

## Purpose
This project was built to practice frontend development skills, focusing on layout structuring, styling, and static website design.
